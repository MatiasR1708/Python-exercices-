# AcadReserve

Autor: Matías Ríos Ospina

Sistema de reservas académicas por consola en Python.

## Ejecución
python main.py

## Archivos
usuarios.csv → login  
espacios.csv → espacios  
reservas.csv → reservas  
configuracion.json → reglas

## Reglas
- Límites por tipo de usuario
- Solo espacios DISPONIBLES
- Control de exceso de tiempo
